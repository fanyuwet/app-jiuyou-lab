# app-jiuyou-lab

📦 一个用于归档与发布独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为多个独立前端页面的集合与版本管理空间。

## 项目简介

`app-jiuyou-lab` 主要用于存放各类独立的 HTML 文件。  
这些页面可能涉及实验性功能、工具页面、静态展示页面等，彼此之间无强关联，各自独立运行。  
仓库本身不承担网站托管或内容推广功能，仅作为页面归档与维护的载体。

## 目录说明

```
├── pages/            # 存放所有独立 HTML 页面文件
├── assets/           # 公共资源文件（如 CSS、JS、图片等）
├── docs/             # 文档或说明文件（可选）
└── README.md         # 本文件
```

- `pages/`：每个子文件夹或文件代表一个独立页面，命名尽量简洁明确。
- `assets/`：存放多个页面共用的样式、脚本或静态资源，避免重复引用。
- `docs/`：可放置与仓库相关的开发说明、变更记录等辅助文档。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可直接在浏览器中打开运行。
- 页面之间不共享状态或数据，不依赖特定后端服务。
- 归档的页面可能处于不同完成阶段（开发中、稳定版、实验版），请以文件内注释或文档说明为准。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来修正问题、改进页面或增加新页面。
- 新增页面请放置在 `pages/` 目录下，并尽量附带简要说明。
- 不建议在页面中包含外部跟踪、统计或营销代码，保持页面纯粹。
- 本仓库不保证所有页面长期可用或兼容最新浏览器版本，请自行评估使用。
